Hardware Service Management System

Multi-tenant service tracking, WhatsApp reminders, GPS photo proof — banks & individual clients

User Roles

  • • Admins — Web dashboard, manage all data
  • • Technicians — Android app, assigned visits only
  • • Customers — WhatsApp recipients (no login)

Key Workflows

  • • Auto-reminders every 6 months via WhatsApp
  • • Field visit: GPS + photo upload as proof
  • • Next service date auto-calculated
  • • Reminder log with delivery status

Tech Stack

  • • Supabase — DB, Auth, Storage, Edge Functions
  • • React + Vite — Admin dashboard (Cloudflare Pages)
  • • React Native — Android field app
  • • WhatsApp Cloud API — Message delivery
  • • pg_cron — Scheduled reminder checks
